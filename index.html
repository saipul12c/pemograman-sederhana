Proyek Pembelajaran Dasar Pemrograman

Memahami dasar-dasar pemrograman dengan fokus pada JavaScript

Deskripsi

Proyek ini dirancang untuk membantu pemula dalam mempelajari dasar-dasar pemrograman dengan fokus pada JavaScript. Struktur proyek ini membagi materi pembelajaran ke dalam beberapa kategori utama untuk memudahkan pemahaman konsep-konsep esensial, seperti variabel, struktur kontrol, fungsi, dan pemrograman berorientasi objek.

Struktur Proyek

Folder src

Folder ini adalah inti dari materi pembelajaran, yang dibagi menjadi beberapa kategori utama:

  • variables: Mencakup konsep variabel dalam JavaScript, termasuk deklarasi, jenis variabel (let, const, var), serta scope dan hoisting.
  • control-structures: Struktur kontrol alur program:
    • if-statements: Penggunaan pernyataan if, else if, dan else.
    • loops: Iterasi menggunakan for, while, do-while.
    • switch: Pengambilan keputusan dengan switch-case.
    • ternary-operators: Penggunaan operator ternary (? :).
    • error-handling: Penanganan error dengan try, catch, finally, throw.
  • functions: Berbagai jenis fungsi di JavaScript seperti basic-functions, arrow-functions, callback-functions, dll.
  • objects: Konsep dasar OOP dalam JavaScript seperti inheritance, polymorphism, dll.

Folder docs

Berisi dokumentasi tambahan, termasuk file README.md dengan panduan umum tentang proyek ini.

Panduan Penggunaan

  1. Eksplorasi folder src untuk memulai pembelajaran.
  2. Ikuti panduan yang ada dalam file README.md di setiap subfolder.
  3. Praktikkan kode yang disediakan untuk memahami konsepnya.
  4. Pelajari lebih lanjut menggunakan dokumentasi tambahan.
